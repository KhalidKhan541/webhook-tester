# Webhook Tester

Test and debug webhooks easily. Inspect incoming HTTP requests in real-time.

## Features
- Real-time request inspection
- Request headers & body display
- HTTP method support
- Copy webhook URL
- Request history
- Mobile-friendly

## Tech Stack
- HTML/CSS/JavaScript
- Cloudflare Pages

## Live Demo
https://webhook-tester.pages.dev

## Run Locally
Open index.html in your browser.
